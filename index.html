About Me

Hello, I was born and raised in Fresno, CA. Early in my career, I had the wonderful opportunity to move and work in Los Angeles, CA. That is where I taught myself basic inline HTML. One of my first projects was a small-scale online radio station. I built an Apache Server to run my radio station automation programs and store my music library along with two additional sites using the help of HTML templets. Fast forward to 2021, I signed up for Bitwise Industries “Websites for beginners” and “Mobile Friendly Websites”. I learned HTML, CSS Styling, Bootstrap and GitHub. Now I'm continuing to expanding my knowledge in website development.
